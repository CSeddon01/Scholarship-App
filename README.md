# The Society of Marth Washington Scholarship Application


## Description 

This scholarship application will allow any orgization or school to present their scholarship award and requirements. Applicants will be able to submit their personal information and upload files such as transcripts , directly on the app which will save in a secure local storage. Scholarship officals will be able to pull the applicant's infromation from local storege and use a grading matrix to properly evaluate the applicate.

## Usage 

This application allows orgizations to present their scholarship and its rewards. School will be able to present their requirements for the scholarship.applicants will be able to apply directly on the application. The application will be saved in a local storage. Evaluators will be able to pull the application from the local storage. The application will then be loaded into a grading matrix. In the grade matrix, evaluators will be able to review and grade each application.
